# hello-go

## Requirements
1. Make
2. Go dev tools
- golint (go get -u golang.org/x/lint/golint)
- gin (go get -u github.com/codegangsta/gin)

## Commands
Ensure that GOPATH and GOBIN are in environment variables

1. Lint 
```
make lint
```

2. Build
```
make build

```

3. Run
```
make run
```

4. Dev mode with hot reload
```
make dev
```
